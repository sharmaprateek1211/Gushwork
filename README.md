# Gushwork Frontend Assignment

## Overview
This project is a responsive frontend webpage developed using Vanilla HTML, CSS, and JavaScript based on the provided Figma design specifications.

The application is designed with a focus on responsiveness, performance, clean UI implementation, and interactive user experience.

---

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

---

## Features

### Responsive Design
- Fully responsive across desktop, tablet, and mobile devices
- Flexbox and CSS Grid based layouts
- Optimized spacing and layout adjustments

### Sticky Header
- Header appears after scrolling beyond the first fold
- Smooth transition and animation effects
- Automatically hides while scrolling back

### Interactive Image Carousel
- Horizontal image carousel implementation
- Previous/Next navigation controls
- Smooth scrolling behavior

### Image Zoom Effect
- Hover zoom interaction on carousel images
- Smooth transition animations

### Performance & Accessibility
- Semantic HTML structure
- Cross-browser compatibility
- Optimized styling and JavaScript
- Accessibility considerations using proper image alt attributes
